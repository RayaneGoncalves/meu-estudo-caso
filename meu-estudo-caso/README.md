# meu-estudo-caso

Projeto entregue para a Atividade Prática: Estudo de Caso com Rotas, Consumo de API e Testes Automatizados.

Estrutura do projeto:
- web/ -> Aplicação React + Vite (consome API de alunos, testes com Vitest)
- mobile/ -> Aplicação Expo + React Native (consome API de alunos)
- README inclui instruções de execução, rotas e testes.

## Escolhas do projeto
- Tema visual: Rosa claro pastel (cards com imagem)
- Dados: Alunos consumidos da API (conforme roteiro)

## API usada
Base URL: https://proweb.leoproti.com.br/api

## Web - Como executar
1. cd web
2. npm install
3. npm run dev
4. npm run test

## Mobile - Como executar
1. cd mobile
2. npm install
3. expo start

## Notas
- O projeto contém código mínimo funcional para execução local.
- Ajuste variáveis de ambiente se necessário.
